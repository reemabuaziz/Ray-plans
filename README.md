# Ray plans

Ray plans is a smart academic planner designed to help students organize their assignments, track progress, and generate study plans automatically.

## ✨ Features
- 📸 Extract assignments from screenshots using OCR
- 🧠 Smart study plan generator based on deadlines
- 📊 Progress tracking system
- 🎨 Clean pastel modern UI
- 📱 Installable as a mobile app (PWA)
- 🔔 Browser notifications support

1. Open the website
2. Tap "Add to Home Screen"
3. Use it like a native app

## 💡 Idea
This project was built to solve a real student problem:
keeping track of assignments from platforms like Blackboard and managing study time efficiently.
